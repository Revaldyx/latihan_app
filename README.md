## 🌦 Weather App - First Release

A simple weather application built using Flutter with clean architecture principles.

### ✨ Features
- Search weather by city name
- Get weather automatically using GPS location
- Current weather data (temperature, windspeed, condition)
- 7-day weather forecast
- Animated weather icons using Lottie
- Error handling (network, server, location, parsing)
- Last searched city saved locally

### 🧠 Tech Stack
- Flutter (Provider State Management)
- Open-Meteo API
- Geolocator & Geocoding
- Clean Architecture (Datasource, Repository, Provider)
- Lottie Animations

### ⚠️ Known Limitations
- Location permission required for GPS feature
- API depends on external service (Open-Meteo)

### 🚀 Future Improvements
- Add hourly forecast
- Improve UI animations
- Add dark mode
- Better error UI experience
